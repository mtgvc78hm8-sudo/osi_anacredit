# osi_anacredit
Python-based AnaCredit OSI tool for SQL data cleaning, aggregation and exposure visualisation

PURPOSE:
This script connects to an OSI Mission SQL database, extracts AnaCredit- style loan tape data, performs data cleaning and enrichment, calculates cumulative debtor/group exposures and collateral coverage, and visualises the Top-20 debtors and groups by exposure.

TECHNOLOGIES:
- Python
- pandas
- sqlalchemy / pyodbc
- matplotlib
